Requires the Assembla Auth Plugin to be used as the authentication source.

Assembla OAuth token is used to identify user for accessing Assembla API.

ADMIN permission is granted to any any authenticated user with Assembla ALL permission (usually space owner)

READ and BUILD job permissions are granted to any authenticated user with Assembla EDIT permission (usually space member)

READ permission is granted to any authenticated user with Assembla VIEW permission (usually space watcher)

It is also available to set Admin usernames list. Those users will have Jenkins ADMIN permission plus basic authorization access to Jenkins API using username/API key pair.
